# Remote Work Health Analysis
This repository analyzes the affect of job roles, salary, and industry on health health outcomes using survey data.

## Repository Structure
- notebooks/ - analysis notebooks
- src/ - reusable Python functions
- data/ - raw and processed(cleaned) datasets

## How to Run
Run notebooks in order:
1. 01_exploration.ipynb
2. 02_cleaning.ipynb
3. 03_analysis.ipynb

## Visualization Approach
Data cleaning and analytical processing are conducted in Python (pandas, numpy). Visualization is performed using R programming (ggplot2). Cleaned datasets were imported from Python into R to create figures
